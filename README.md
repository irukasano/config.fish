# config.fish

## install

### install fish shell

    yum install fish --enablerepo=epel

### install fzf

    git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
    ~/.fzf/install

### install fish config(this repo)

    mkdir ~/.config
    cd ~/.config
    git clone https://github.com/irukasano/config.fish.git
